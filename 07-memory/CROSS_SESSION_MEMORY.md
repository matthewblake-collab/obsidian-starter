## How to use this file

At the end of any significant AI session, ask yourself:
> *"What rule or pattern did I learn today that I should always apply going forward?"*

Write it here. Be specific. "Be more careful" is useless. "Supabase RLS-blocked DELETEs return success with 0 rows — always verify with a count check" saves hours.
