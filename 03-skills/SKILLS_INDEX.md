## How to write a skill note

```
# Skill: [Name]

## When to use this skill
[Exact trigger — when should this be invoked?]

## What you need before starting
[Any inputs, files, or setup required]

## Steps
1. [Step one — specific, no assumed knowledge]
2. [Step two]
3. [Step three]

## Rules — never break these
- [Rule 1]

## Example output
[Show what good looks like]
```
